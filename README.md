# DiosesDrakes

A custom mythology and divine favors plugin for **DrakesCraft Network**.

## Features

- ⚡ **Divine Altars:** Players can build altars and pray to custom gods for temporary buffs.
- 📜 **Blessings & Curses:** Dynamic event-driven rewards or penalties based on player actions.
- 🛠️ Highly configurable via YAML files.

## Compatibility

- Minecraft Purpur 1.21.11 (Paper API)
